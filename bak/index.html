What

Beehive is an open-source application deployment framework. It's goal is to provide a stable, scalable, fault-tolerant application in an unstable environment. Written in Erlang and Ruby, Beehive provides a distributed router that sits in front of a mesh of client instances.

 

Distributed

From the ground-up, Beehive has been designed to be fault tolerant and distributed. This means that beehive will work the same on one server as it will on 100. Written primarily in Erlang, Beehive is natively scalable.

 

Open-source

Beehive is 100% open-source software and has a growing community. It supports the open-source community and was written to work on any hardware. It's extensible and modular.
